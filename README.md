# MediaDeck

Your personal media collection in one place. Built with **Kotlin** and **Jetpack Compose**, it combines comics, photos, and videos.

## Features

-  Comic Reader
-  Photo Gallery
-  Movie Library
-  SMB Network Support


## Built With

- Kotlin
- Jetpack Compose
- Room Database
- Media3 / ExoPlayer
- Coil
- Material Design 3

## Requirements

- Android 8.0 (API 26) or higher
